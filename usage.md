### hexo 发布文章使用方法
```
1) $ hexo new [layout] <title> OR $ hexo n
2) 编辑保存文章
3) $ hexo g or $ hexo generate
4) $ hexo d or $ hexo deploy
5) $ hexo s or $ hexo server
```
